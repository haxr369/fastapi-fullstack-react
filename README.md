# fastapi-fullstack-react
University of Seoul CVLAB's Livinglab

# 개발목표
사용자가 사이트에 접속해서 쉽게 식물 분류 모델을 사용할 수 있도록 한다.
Frontend는 React를 사용하고, Backend는 Fastapi를, 그리고 DB는 Mysql을 사용한다.
